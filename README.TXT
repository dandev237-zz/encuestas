Código para resolver el problema de las encuestas con un enfoque divide y vencerás.

Enunciado: Se han realizado el mismo número de encuestas en dos ciudades diferentes, habiéndose ordenado las encuestas 
en cada una de las ciudades alfabéticamente por el nombre de la calle. Se desea obtener un algoritmo Divide y Vencerás
que en un tiempo O(log n) obtenga el nombre de la calle que está en la mediana de las dos ciudades.
